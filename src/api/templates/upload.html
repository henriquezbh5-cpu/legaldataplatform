LegalDataPlatform - Self-service ingest

Upload a CSV to ingest

Drag-and-drop or browse. The system detects the schema and runs the pipeline. Results appear below.

What this does: the API reads your CSV headers, fuzzy-matches them against the 3 known schemas (legal_documents, counterparties, transactions), and triggers the corresponding Prefect flow. The raw file lands in data/inbox/ and the cleaned data ends up in PostgreSQL.
